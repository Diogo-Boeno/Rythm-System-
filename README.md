# Rhythm_System

Roblox project by Cosmo.

## Setup

1. Install Aftman: https://github.com/LPGhatguy/aftman
2. Run `aftman install` in project root
3. Run `wally install` to fetch dependencies
4. Run `rojo serve` and connect via Roblox Studio plugin

## Structure

- `src/ReplicatedStorage/Shared/Framework/` — core framework
- `src/ServerScriptService/Services/` — server-side services
- `src/StarterPlayerScripts/Controllers/` — client-side controllers
